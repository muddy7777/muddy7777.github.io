---
title: 研究任务 Prompt
description: 用于文献检索、技术调研和论文相关任务的 prompt 模板。
---

```text
请围绕以下主题做研究任务：

研究问题：
{你真正想回答的问题}

范围：
{时间范围、学科范围、应用场景}

来源要求：
- 优先论文、官方文档、标准、权威报告。
- 标明 DOI、arXiv、会议或期刊信息。
- 区分事实、作者观点和你的推断。

输出：
1. 结论摘要
2. 关键论文或资料表格
3. 方法对比
4. 争议或不确定性
5. 后续阅读路线

约束：
不要编造引用。
如果信息可能过时，请联网核查。
```

## 适用场景

- 想快速进入一个新研究方向。
- 需要比较多个方法、模型或系统。
- 需要为论文写 related work 或实验设计做准备。
