title: 让 Agent 读取本站的协议
description: 把本站作为外部上下文交给 agent 时的推荐方式。
---

当你希望 agent 使用本站内容时，不要只丢首页。更好的方式是明确入口、范围和任务目标。

## 推荐指令

```text
请先读取 https://muddy7777.github.io/llms.txt。
根据我的任务选择相关页面，不要无差别抓取整站。
如果你需要完整核心上下文，再读取 https://muddy7777.github.io/llms-full.txt。
读取后请用自己的话总结你将采用的工作方式，再开始执行。
```

## 按任务追加范围

```text
本次任务是代码实现，请重点读取 Skills、Prompts 中与 coding 和 code-review 相关的内容。
```

```text
本次任务是研究调研，请重点读取 research prompt 和研究类 skill 的说明。
```

```text
本次任务是配置工具，请重点读取 MCP 栏目，并注意不要要求我公开任何密钥。
```

## 验收方式

让 agent 在开始前回答：

1. 你读取了哪些页面？
2. 你认为本次任务应该遵循哪些约束？
3. 你会产出什么？
4. 你会如何验证？

如果这四点说不清，说明上下文还没有被正确吸收。
