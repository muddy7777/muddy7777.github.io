---
title: Skill 卡片模板
description: 新增个人 skill 笔记时可复制的结构。
---

复制下面模板，新建一篇 skill 页面即可。

```md
---
title: Skill 名称
description: 一句话说明它解决什么问题
---

# Skill 名称

## 适用场景

## 不适用场景

## 输入要求

## 推荐流程

## 输出格式

## 验证方式

## 常见失败模式

## 示例

## 相关页面
```

## 写作标准

- “适用场景”和“不适用场景”都要写，减少误触发。
- “验证方式”必须具体，可以是命令、截图、渲染、测试、引用核查。
- 示例要能被 agent 直接复用，不只写抽象原则。
