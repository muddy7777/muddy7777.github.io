# Codex 常用 Skills 清单

## 开发与验证

- browser control：本地网页预览、点击、截图、交互验证。
- jupyter-notebook：创建或编辑实验 notebook。
- plugin-creator：创建 Codex 插件。
- skill-creator：创建或更新 skill。

## 内容与文件

- documents：Word、docx、文档排版和审阅。
- spreadsheets：xlsx、csv、表格分析和图表。
- presentations：PPTX、幻灯片和演示稿。
- pdf：PDF 阅读、生成、版式检查。
- imagegen：生成或编辑位图视觉素材。

## 研究与写作

- literature-search：找论文、做阅读列表、比较方法。
- academic-research-suite：深度研究、系统综述、论文流程。
- scientific-writing：摘要、引言、方法、实验、结论润色。
- research-paper-workflow：实验计划、论文里程碑、审稿回复。

## 产品与安全

- openai-docs：OpenAI API、模型、Codex 使用问题。
- security-best-practices：明确要求安全最佳实践或安全审查。
- security-threat-model：明确要求威胁建模。

## 选择规则

1. 用户明确点名 skill 时，优先使用该 skill。
2. 任务明显匹配 skill 描述时，使用最小必要 skill 集合。
3. 普通实现或调试先读代码库，不要机械套 skill。
4. 涉及当前事实或高风险信息时，先核查来源。
